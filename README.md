# SchoolSafe — The Wise School International

Application de gestion scolaire intelligente (PWA) — Kinshasa, RDC.

## Accès
**`https://medt121.github.io/zalavrai/`**

## Fonctionnalités
- Présence par QR code
- Sécurité des enfants (scanner entrée/sortie)
- Suivi des résultats et palmarès
- Communication parents-école
- Gestion des paiements scolaires (frais scolaires, rattrapages)
- Cahier de texte numérique
- Notifications en temps réel
- Rapports journaliers caisse

## Rôles
- Direction 1 — accès complet
- Direction 2 — accès complet sauf financier
- Direction 3 (Caisse) — gestion paiements journaliers
- Enseignant — présences, notes, devoirs, cahier de texte
- Gardien — scanner entrée/sortie
- Parent — suivi enfants, paiements, messages

## Technologies
- PWA (Progressive Web App) — Single HTML file
- Supabase (base de données en ligne)
- GitHub Pages (hébergement)
