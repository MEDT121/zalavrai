      ② Le nommer ZALAVRAI-DB

      ③ Cliquer Partager → Toute personne ayant le lien → Lecteur

      ④ Copier l'URL de la barre d'adresse et la coller dans le champ ci-dessus

      

        ℹ️ Les 7 onglets (utilisateurs, clients, ventes…) seront créés automatiquement via GAS

Activer l'écriture (recommandé)

        
Sauvegarde automatique vers Sheets

      

    

    

      Dans votre Google Sheets (ZALAVRAI-DB) :

      ① Extensions → Apps Script

      ② Effacer le code → Coller le Code.gs fourni

         Le SHEET_ID est déjà configuré dans le Code.gs ✅

      ③ Déployer → Nouvelle appli web

         • Exécuter en tant que : Moi

         • Accès : Tout le monde

      ④ Coller l'URL de déploiement ci-dessus

      ⑤ Cliquer ☁️ Connecter
      

        ⚠️ Mises à jour du code GAS :

        → Déployer → Gérer les déploiements → ✏️ Modifier

        → JAMAIS "Nouveau déploiement" (l'URL changerait !)
